# Publish Checklist

## Before pushing to GitHub
- set the repository name to `ura-cc`
- add the short repository description from `docs/seo-keywords.md`
- set GitHub topics
- upload a social preview image
- pin the repository if it becomes a flagship project
- verify README formatting on GitHub
- confirm LICENSE year/name
- create initial tags or milestone labels if desired

## Recommended GitHub topics
remote-desktop, remote-control, webrtc, rust, flutter, windows, android, mouse-control, keyboard-control, mobile-app

## Suggested first release title
`v0.1.0-docs-foundation`

## Suggested release notes summary
- locked product direction
- locked MVP scope
- added architecture and security docs
- added community and issue templates
- prepared repo for implementation scaffolding
