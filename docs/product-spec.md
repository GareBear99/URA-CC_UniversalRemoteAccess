# Product Spec

## Name
URA-CC — CouchController

## Expansion
Universal Remote Access

## Mission
Provide a clean, one-handed, couch-friendly way to control a computer from a phone with live view, fast clicks, and lightweight text entry.

## Core UX promise
URA-CC should feel more like using a controller and less like fighting a tiny trackpad.

## Primary users
- people controlling a TV-connected computer
- creators controlling a media machine from across a room
- users who want one-handed mouse + keyboard access from a phone
- anyone who wants a lighter, simpler alternative to full remote desktop workflows

## Primary wedge
Phone-to-PC control optimized around a thumb-dial cursor model.

## MVP
- Windows host
- Android client
- LAN-only setup
- live desktop video
- dial cursor movement
- left click / right click
- keyboard text input
- trusted-device approval
